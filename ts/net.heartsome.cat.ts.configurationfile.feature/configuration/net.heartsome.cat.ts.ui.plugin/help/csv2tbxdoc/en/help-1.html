Step 1: Import a CSV file

The first task you should do is open the CSV file you wish to convert. You can start this process in two ways:

  1. Use the option "Open CSV File" from File menu

  2. Click on button

Either method opens the following dialog window:

Open CSV File

Type the name of the CSV file you wish to import or use the Browse button to locate it.

Select the appropriate values for each field and click on Accept button. Check the Glossary for more information on the meaning of each field.
