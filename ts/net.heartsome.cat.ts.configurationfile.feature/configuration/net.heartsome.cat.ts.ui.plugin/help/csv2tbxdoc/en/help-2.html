Step 2: Remove Unnecessary Columns

After completing Step 1, the selected CSV file is loaded by the program and its content is displayed in the main screen.

CSV File Loaded

Check the columns displayed by the program and if any contains unnecessary data, click on (or select "Remove Columns" in Tasks menu) to display the following dialog:

Remove Columns

Select the columns that you wish to remove and click on Remove Selected button. To select more than one column, keep Ctrl key pressed while you click on each column label.

After you are done removing unwanted columns, click on Accept button to return to the main window.
