Step 3: Select Column Properties

Before saving the CSV file as TBX, you must set the properties of each column. To do so, click on (or choose "Column Properties" in Tasks menu) to display the following dialog:

Select Column Properties

The labels shown at the lef of the drop down selectors are the same labels displayed at the top of each column.

A set of drop-down boxes allows you to map columns to concepts or terms and their attributes. The following table sumarises the available options:

After selecting the properties for each column click on Accept button to return to the main window.

Column properties can be saved and reused when converting CSV files with similar structure. Use the Save Configuration button to store selected settings on disk and the Load Configuration button to recover previoulsy saved settings for reuse.
