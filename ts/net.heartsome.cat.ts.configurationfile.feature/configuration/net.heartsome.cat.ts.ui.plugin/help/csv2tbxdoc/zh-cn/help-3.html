第 3 步：选择列属性

在您将 CSV 文件另存为 TBX 文件之前，您必须为各列设置正确的属性。 单击 或到任务菜单选用"列属性"以便显示下示对话框:

选择列属性

各列所显示的值就是各对应列的标签。
