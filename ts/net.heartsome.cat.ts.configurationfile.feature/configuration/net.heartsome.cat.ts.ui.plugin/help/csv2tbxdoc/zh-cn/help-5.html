词汇表

  • CSV (逗号分隔值)

    将数据以列的格式列表的纯文本文件，值与值之间采用逗号为分隔符号。每一行代表一套数据。

  • TBX

    TBX 是为创建并保存术语数据而设计的 XML 开放标准格式。作为一种开放标准，TBX 需要各术语软件的全面支持，如果大家都采用同一套标准，用户便可免除许多不兼容的问题。欲知详情，请访问http://www.lisa.org/tbx。

  • 列分隔符

    用于分隔数据列的字符。 一般是采用逗号。

  • 文本分隔符号

    用于分隔数据列内容的字符。 一般是采用双引号。需要适用分隔字符的情况一般是在列内文字含有作为列分隔符的字符时。

  • 字符集

    计算机系统用于代表指定国家或地方的字符(字母、数字、标点符号等等)的数字代码。Unix系列的操作系统(Linux和Mac)都以UTF-8为默认字符集。 Microsoft Windows 则使用该公司独家拥有的 Windows-XXX 系列字符集。 许多欧洲国家则采用取名 ISO-8859-1 (ISO Latin-1) 的 ASCII 码扩展字符集。

  • 主语言

    全部概念中用于输入备注和属性的语言。

  • XCS 模板

    XCS 模板是一个 XML 文件，用于列表可为各个术语指定的属性。其实，这种模板是一份数据限制规格。
