Step 3: Select Language Codes

Before saving the CSV file as TMX, you must select the language codes for each column. To do so, click on (or choose 'Select Languages" in Tasks menu) to display the following dialog:

Select Languages

The labels shown at the lef of the drop down selectors are the same labels displayed at the top of each column.

After selecting the desired language codes click on Accept button to return to the main window.
